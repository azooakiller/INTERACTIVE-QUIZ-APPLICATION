# JS Mastery Quiz — Project Description

## What Is This Project?

JS Mastery Quiz is a simple web-based quiz application built using three core web technologies — HTML, CSS, and JavaScript. The goal of this project is to test a user's knowledge of JavaScript through a set of multiple-choice questions. The quiz runs entirely in the browser, which means no server, no database, and no internet connection is needed after the page loads. Anyone can open the HTML file and start playing immediately.

This project is perfect for beginners who are learning JavaScript and want to test themselves, or for teachers and trainers who want a ready-made quiz tool they can customize and share.

---

## What Technologies Are Used?

This project uses only three files, each with a specific job:

**HTML (index.html)** — This is the structure of the page. It defines what elements exist on the screen, like the title, the question box, the answer buttons, the progress bar, and the result screen. Think of HTML as the skeleton of the application.

**CSS (style.css)** — This controls how everything looks. It sets the colors, fonts, sizes, spacing, and animations. The purple gradient background, the white card in the center, the green correct button, the red wrong button — all of that comes from CSS. Think of CSS as the paint and decoration on top of the skeleton.

**JavaScript (script.js)** — This is the brain of the application. It controls everything that happens when the user interacts with the quiz. It loads the questions, shuffles them randomly, starts the timer, checks if the answer is correct, updates the progress bar, and shows the final score. Without JavaScript, the quiz would just be a static page with no behavior.

---

## How Does the Quiz Work?

When the page loads, JavaScript automatically starts the quiz. It takes all 20 questions from the question bank, shuffles them into a random order, and picks the first 10 to use for that round. This means every time you play, you get a different set of questions in a different order, keeping the quiz fresh.

Each question is displayed one at a time with four answer choices. The answer choices are also shuffled randomly so the correct answer is never in the same position twice.

Once a question appears, a 30-second countdown timer begins. The user must pick an answer before the time runs out. If they answer in time, the selected button turns green (correct) or red (wrong), and the correct answer is always highlighted so the user can learn from their mistake. If the timer hits zero before the user answers, the correct answer is shown automatically and the quiz moves on.

After the user answers, a "Next" button appears. Clicking it loads the next question. This continues until all 10 questions are done.

At the end, a result screen appears showing the user's score out of 10 and a short motivational message depending on how well they did.

---

## What Features Does It Have?

- **20 questions in the bank** — Topics include variables, data types, array methods, events, closures, strict mode, JSON, and more.
- **Random shuffle every round** — Both the questions and the answer options are shuffled each time, so no two games are exactly the same.
- **30-second timer per question** — Adds a challenge and keeps the quiz moving at a good pace. The timer turns red when only 10 seconds remain as a warning.
- **Instant feedback** — The correct answer is always shown after the user responds, which helps learning and not just scoring.
- **Progress bar** — A purple bar at the top grows with each question so the user always knows how far along they are.
- **Final score and message** — After all 10 questions, the score is displayed along with a message: excellent, good effort, or keep practising — depending on the percentage.
- **Play Again button** — Reloads the page so a fresh, reshuffled quiz starts immediately.

This project is an excellent learning example because it covers many important JavaScript concepts in one place — arrays, objects, functions, DOM manipulation, event listeners, timers with `setInterval`, and conditional logic. The code is written in plain, readable JavaScript with no frameworks or libraries, so beginners can follow every line without needing to learn anything extra.

Every section of the code is clearly commented and explained, making it easy for anyone to read through and understand exactly what each part does and why it is there.
